# Postman API Testing Project

## Description
This project demonstrates API testing using Postman with CRUD operations and API chaining.

## Features
- GET, POST, PUT, DELETE request testing
- API chaining using environment variables
- Response validation using test scripts
- Status code and response time validation

## Tools Used
- Postman
- REST API
- JavaScript (pm object)

## How to Run
1. Import the collection JSON file into Postman
3. Run the collection using Collection Runner

## Project Highlights
- Implemented API chaining
- Used dynamic variables
- Validated API responses
